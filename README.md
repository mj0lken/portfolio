# [mj0lken | Terminal](https://mj0lken.xyz)

A terminal style portfolio site.

![screenshot](/docs/example-view.png)

## Quick Start

### Using npm/yarn

1. Install dependencies:

  ```bash
  npm install
  ```

2 Run the project locally in dev mode:

  ```bash
  npm run dev
  ```

3 Build the project:

  ```bash
  npm run build
  ```

4 Run the server:

  ```bash
  npm run start
  ```
